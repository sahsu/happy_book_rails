## 查看方法

1. 安装 nvm  (http://siwei.me/blog/posts/nvm-node )
2. 安装 node (推荐node 10版本。其他更高版本貌似会报错）
3. 安装 gitbook  (参考: http://siwei.me/blog/posts/gitbook-gitbook )

```
$ nvm use 10 ( 本步骤可选 )
$ npm install gitbook-cli -g
```

4. 运行：

```
$ cd src
$ gitbook serve  ( 注意这里不是 $ gitbook server)
```

然后就可以看到了!
