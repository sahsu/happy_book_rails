## 查看方法

1. 安裝 nvm  (http://siwei.me/blog/posts/nvm-node )
2. 安裝 node (推薦node 10版本。其他更高版本貌似會報錯）
3. 安裝 gitbook  (參考: http://siwei.me/blog/posts/gitbook-gitbook )

```
$ nvm use 10 ( 本步驟可選 )
$ npm install gitbook-cli -g
```

4. 運行：

```
$ cd src
$ gitbook serve  ( 注意這裏不是 $ gitbook server)
```

然後就可以看到了!
